# Содержание отчета по индивидуальному заданию

Отчет должен содержать следующую информацию.

1. Фамилию, Имя, Отчество исполнителя.
2. Группу.
3. Вариант задания и условие задачи.
4. Тесты, демонстрирующие проверку разработанных программ.
5. Результаты тестовых прогонов для различных исходных данных.
6. Тексты программы на языке ассемблера.
7. Дополнительную информацию, подтверждающую выполнение задания в соответствие требованиям на предполагаемую оценку.
